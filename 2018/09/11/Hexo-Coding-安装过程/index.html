配置 Git

  • 本地生成 SSH 公钥
    ssh-keygen -t rsa -C "你的邮件地址"

  • 用户信息配置
    git config --global user.name "John Doe"
    git config --global user.email johndoe@example.com

  • 安装 Push 工具
    npm install hexo-deployer-git --save

ps：SSH 公钥文件一般存储在 C:\Users\用户\.ssh。

配置 Coding

登录 coding，创建新项目，需要注意的是项目名字一定得是: 你的用户名。

  • 添加公钥到 Coding
    • 根据上一步的提示，找到SSH公钥文件（默认为id_rsa.pub），用记事本打开，全选并复制。
    • 登录 Coding，右上角 头像 -> 个人设置 —> SSH公钥 —> 新增公钥。把公钥粘贴到公钥内容中，填好title并点击添加。
    • git bash中输入命令$ssh -T git@git.coding.net，选yes，检测公钥是否添加成功。
